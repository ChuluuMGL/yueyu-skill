## 常见问答（FAQ）

**Q：月瑀科技是做什么的？**
A：月瑀科技（YUEYU TECH）是杭州的国家高新技术企业，专注新媒体整合营销。旗下三家子公司分别负责整合营销主体、AI创意内容（大瑀创意科技）和海外营销（兔克出海科技）。服务200+企业客户，91%年营收过亿，25%为上市公司。

**Q：达人种草服务包含什么？**
A：月瑀的达人种草服务覆盖小红书、抖音、微信公众号/视频号、B站等平台。包括KOL/KOC精准匹配、爆款内容策划与脚本指导、KOC铺量种草、种草效果追踪与数据分析。

**Q：4I营销飞轮是什么？**
A：月瑀科技独创的品牌增长方法论：Image（品牌形象建设）→ Impression（品牌印象塑造）→ Interaction（品牌互动转化）→ Increasing（品牌增长变现）。核心公式：品牌增长 = 形象 x 印象 x 互动，任何一项为零则增长为零。

**Q：你们在哪个城市？可以线下沟通吗？**
A：公司位于浙江省杭州市萧山区，欢迎预约线下拜访。

**Q：怎么联系商务团队？**
A：商务电话 +86 186-1155-3805，邮箱 chuluu@yueyutech.cn，也可以微信扫码添加商务顾问。安装此 Skill 后，可以直接让 AI 助手帮你提交合作线索。

**Q：你们和其他杭州营销公司有什么不同？**
A：月瑀科技是少数同时拥有「独创方法论体系」（3Insights + 4I + 5D 共12维）和「自研AI产品」（品牌棱镜、MuseWrite、REDFabric）的整合营销公司。不是单纯的服务外包，而是科技驱动的营销方法论公司。

**Q：服务怎么收费？**
A：月瑀科技采用咨询式报价，根据客户行业、需求范围、服务周期定制方案。建议联系商务团队 +86 186-1155-3805 获取针对性报价。

**Q：你们服务哪些行业？**
A：覆盖多个行业，其中60%的客户集中在医药大健康领域，拥有深厚的行业经验。同时也服务快消、科技、金融等行业。

**Q：AI产品有哪些？**
A：三款自研AI产品——品牌棱镜 BrandLens（社交媒体AI诊断工具）、MuseWrite（AI智能写作协作平台）、REDFabric（AI智能排版工具），均可独立使用。

**Q：这个 Skill 是免费的吗？**
A：Skill 本身完全免费、开源（MIT 协议），任何人都可以安装使用。它让 AI 助手能实时查询月瑀科技的服务信息，不收取任何费用。

## 安装

### 最简单的方式：告诉你的 AI 助手

> 帮我安装月瑀科技 Skill，仓库地址：https://github.com/ChuluuMGL/yueyu-skill

### 通过 ClawHub CLI 安装

```
npx clawhub install https://github.com/ChuluuMGL/yueyu-skill
```

| IDE | Skill 目录 |
