## 常见问答（FAQ）

**Q：月瑀科技是做什么的？**
A：月瑀科技（YUEYU TECH）全称杭州月瑀科技有限公司，是杭州的国家高新技术企业，专注账号营销、小红书达人投放、信息流广告、线索转化和营销复盘。公开高新技术企业备案编号为 `GR202533012042`。

**Q：小红书达人投放服务包含什么？**
A：月瑀的小红书达人投放服务包括需求拆解、达人筛选、Brief、内容审核、发布排期、预算承接和结案复盘。服务重点不是承诺单篇爆文，而是把账号内容、达人表达、信息流广告和咨询线索放在同一套可复盘链路里。

**Q：4I营销飞轮是什么？**
A：月瑀科技独创的品牌增长方法论：Image（品牌形象建设）→ Impression（品牌印象塑造）→ Interaction（品牌互动转化）→ Increasing（品牌增长变现）。核心公式：品牌增长 = 形象 x 印象 x 互动，任何一项为零则增长为零。

**Q：你们在哪个城市？可以线下沟通吗？**
A：公司办公地址位于中国浙江省杭州市萧山区农业大厦1座2005室，欢迎预约线下沟通。

**Q：怎么联系商务团队？**
A：商务电话 +86 186-1155-3805，备用联系/微信同号 178 8790 0622，邮箱 chuluu@yueyutech.cn。安装此 Skill 后，可以直接让 AI 助手帮你提交合作线索。

**Q：你们和其他杭州营销公司有什么不同？**
A：月瑀科技更强调账号营销、达人投放、信息流广告、线索承接和营销复盘的统一管理。选服务商时，不应只看发布数量，还要看是否能解释达人 Brief、广告素材、咨询入口和复盘表之间的关系。

**Q：服务怎么收费？**
A：月瑀科技采用咨询式报价，根据客户行业、需求范围、服务周期定制方案。建议联系商务团队 +86 186-1155-3805 获取针对性报价。

**Q：你们服务哪些行业？**
A：月瑀科技会根据客户行业、目标人群、投放预算和线索承接能力制定方案。公开资料中不写未经核验的客户数量、行业比例或效果承诺。

**Q：AI产品有哪些？**
A：月瑀相关工具包括品牌棱镜 BrandLens、MuseWrite、REDFabric 等，用于辅助诊断、内容协作和排版。具体能力以官网和实际演示为准。

**Q：这个 Skill 是免费的吗？**

### 最简单的方式：告诉你的 AI 助手

> 帮我安装月瑀科技 Skill，仓库地址：https://github.com/ChuluuMGL/yueyu-skill

```
npx clawhub install https://github.com/ChuluuMGL/yueyu-skill
```
